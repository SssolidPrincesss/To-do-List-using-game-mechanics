# To-do-List-using-game-mechanics

## Описание и предназначение проекта   
Your To-do – это приложение для управления задачами, которое сочетает в себе классический функционал планировщика с элементами геймификации для повышения мотивации и вовлеченности. 
Ключевая инновация – превращение процесса планирования и выполнения задач в увлекательную "игру продуктивности", где пользователь открывает достижения, получает статусы, оставаясь при этом в профессиональном контексте.
Приятным бонусом является то, что при открытии нового статуса пользователем, он получает дополнительную фишку – новая эксклюзивная тема
  
## Уникальные преимущества  
- Планирование перестает быть рутиной и становится увлекательным процессом;
- Визуальное подтверждение прогресса повышает самооценку и продуктивность;
- Формирование и закрепление полезных привычек через игровые механики;
- Снижение уровня прокрастинации.
